## Sintaxis de MarkDown:

Listado de elementos básicos de Markdown.

### Cabeceras:

# H1

La cabecera más grande se escribe como:

`# H1`

Permite escribir el titulo con el mayor tamaño posible.


## H2

La segunda cabecera más grande se escribe como:

`## H2`

Permite escribir el titulo con el segundo mayor tamaño posible.


### H3

La tercera cabecera más grande se escribe como:

`### H3`

Permite escribir el titulo con el tercer mayor tamaño posible.


#### H4

La cuarta cabecera más grande se escribe como:

`#### H4`

Permite escribir el titulo con el cuarto mayor tamaño posible.


##### H5

La quinta cabecera más grande se escribe como:

`##### H5`

Permite escribir el titulo con el quinto mayor tamaño posible.


###### H6

La sexta cabecera más grande se escribe como:

`###### H6`

Permite escribir el titulo con el sexto mayor tamaño posible, también es el tamaño más pequeño para titulos.

### Negrita

La negrita permite resaltar el texto con un color más oscuro de lo habitual.

Sintaxis:

` texto sin negrita **texto con negrita** texto sin negrita`

Ejemplo:

texto sin negrita **texto con negrita** texto sin negrita


### Cursiva

La cursiva modifica las palabras para darle un grado de inclinación, esto permite hacer referencias en el texto diferenciandolo de las demás palabras.

Sintaxis:

`texto sin cursiva *texto con cursiva* texto sin cursiva`

`texto sin cursiva _texto con cursiva_ texto sin cursiva`


Ejemplo:

texto sin cursiva *texto con cursiva* texto sin cursiva

texto sin cursiva _texto con cursiva_ texto sin cursiva

### Tachado

Permite tachar el texto para indicar que es información desechada o errónea.

Sintaxis:

`~~Texto tachado~~`

Ejemplo:

~~Texto tachado~~

### Cursiva en negrita y anidada

Permite resaltar aún más algo en un texto.

Sintaxsis:

`**Este texto es _extremadamente_ importante**`

Ejemplo:

**Este texto es _extremadamente_ importante**

### Todo en negrita y cursiva

Resaltamiento en negrita y cursiva:

Sintaxis:

`***Texto resaltado en negrita y cursiva***`

Ejemplo:

***Texto resaltado en negrita y cursiva***

### Cita de Texto

Permite añadir una cita o ejemplo a la explicación.

Sintaxis:

Texto que no es una cita
`> Texto que es una cita`

Ejemplo:

Texto que no es una cita
> Texto que es una cita

### Código de cita

Permite escribir código sin que realice su función en el mensaje.

Sintaxis:

``> Texto que quieres evitar realizar``

Ejemplo:

`>Texto que quieres evitar realizar`

### Enlaces

Permite añadir un enlace en el texto.

Sintaxis:

`Fichero realizado para la [actividad 13](https://aules.edu.gva.es/fp/mod/assign/view.php?id=2440784)`

Ejemplo:

Fichero realizado para la [actividad 13](https://aules.edu.gva.es/fp/mod/assign/view.php?id=2440784)

### Imágenes

Permite añadir una imagen al texto.

Sintaxis:

`![Skadi girando](https://c.tenor.com/m4bGs6iHuSwAAAAC/arknights-spinning.gif)`

Ejemplo:

![Skadi girando](https://c.tenor.com/m4bGs6iHuSwAAAAC/arknights-spinning.gif)

### Listas

Permite hacer una lista en el texto.

Sintaxis lista sin ordenar:

```
- Linea 1
- Linea 2
- Linea 3
```

Ejemplo lista sin ordenar:

- Linea 1
- Linea 2
- Linea 3

Sintaxis lista ordenada:

```
1. J1
2. J2
3. J3
```

Ejemplo lista ordenada:

1. J1
2. J2
3. J3

### Listas Anidadas

Permite realizar varias lineas de listas con diferentes niveles.

Sintaxis:

```
1. Primera linea del primer nivel
    - Primer elementos del segundo nivel
        - Primer elemnto del tercer nivel 
```

Ejemplo:

1. Primera linea del primer nivel
    - Primer elementos del segundo nivel
        - Primer elemnto del tercer nivel 



### Lista de Tareas

Permite crear una lista de tareas dentro del texto

Sintaxis:

```
- [x] Archivo creado
- [ ] Ejercicio Terminado
- [ ] \(Opcional) Detallar información
```

Ejemplo:

- [x] Archivo creado
- [ ] Ejercicio Terminado
- [ ] \(Opcional) Detallar información 

### Notas al pie

Permite crear notas al pie de página en el texto.
Sintaxis:

```
Texto con una nota al pie. [^1]
Otro Texto con una nota al pie. [^2]
Texto con una nota al pie. [^notaza]

[^1]: Descripción de la nota al pie.
[^2]: Otra descripción de la nota al pie.
[^notaza]: Descripción de la notaza al pie.
```

Ejemplo:

Texto con una nota al pie. [^1]

Otro Texto con una nota al pie. [^2]

Texto con una nota al pie. [^notaza]

[^1]: Descripción de la nota al pie.

[^2]: Otra descripción de la nota al pie.

[^notaza]: Descripción de la notaza al pie.
